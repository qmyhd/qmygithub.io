Mapping Project

Recently, I have become very interested in mapping software and algorithms, so I decided to make my own shortest path implementation of Dijkstra's algorithm. Written in entirely C++, this project features a linear congruential random number genrator that creates data for nodes. Each node represents a specific city, and is positioned by coordinates on a plane. To make dijkstra's algorithm run quickly and efficiently, I created an implementation of a min pairing heap to use as a priority queue. The terminal interface features a menu for easy program input, execution, and response. The names of the cities in the program are inspired by my favorite chilhood game, KingsIsle's Wizard101.
